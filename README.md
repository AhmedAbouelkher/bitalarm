# bitalarm
A cryptocurrency tracker and portfolio app, built with Flutter and Dart
[Screenshots can be found here](https://imgur.com/a/3N5mb)


## Getting Started

1. Install Flutter
2. Clone the repo
3. Run `flutter run` (make sure to have an emulator running.
